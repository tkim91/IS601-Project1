Pylint

What is Pylint?

Pylint is a source-code, bug, and quality checker for the Python programming language. It is named following a common convention in Python of a "py" prefix, and a nod to the C programming lint program. It includes the following features:

  • Checking the length of each line
  • Checking the variable names are well-formed according to the project's coding standard
  • Checking that declared interfaces are truly implemented
